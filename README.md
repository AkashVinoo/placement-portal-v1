# placement-portal-v1

## Overview

The Placement Portal is a full-stack web application designed to streamline campus placement activities by providing a centralized platform for students, recruiters, and administrators.

The application enables job posting, student profile management, application tracking, and placement coordination through a structured and user-friendly interface.

This project was developed as part of an application development project coursework and demonstrates the implementation of a database-driven web application with role-based functionality.

## Features

* Student registration and profile management
* Recruiter and company management
* Job posting and application tracking
* Administrative dashboard
* Database integration and data management
* Role-based access control
